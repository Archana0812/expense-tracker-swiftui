
# Expense Tracker – SwiftUI 📊

A simple and clean **Expense Tracker iOS application** built using **SwiftUI**.  
This app helps users track daily expenses, categorize them, and view monthly summaries.

---

## ✨ Features

- ➕ Add new expenses (title, amount, category, date)
- 🗑️ Delete expenses
- 📅 Monthly expense summary
- 💰 Total expense calculation
- 📱 Modern SwiftUI interface
- 🔄 Data persistence using local storage
- 🧪 Unit-tested core logic

---

## 🛠 Tech Stack

- **Language:** Swift
- **Framework:** SwiftUI
- **Architecture:** MVVM
- **Storage:** JSON / UserDefaults (can be extended to CoreData)
- **Testing:** XCTest
- **Platform:** iOS
